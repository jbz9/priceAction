# 做空方法论

> 做空在 Brooks 体系里**比做多难一倍**——不是劝退，是设定正确预期。
> 本文档把【对话沉淀 + Brooks 经典做空 setups + 外汇做空特殊检查 + 心理与纪律】
> 一次性整合成可执行手册。

---

## 一、系统目标与边界

### 1.1 能做什么

+ 在确立的下跌趋势中识别 L1/L2 setup 并做空
+ 在 trading range 上沿 fade
+ 在楔形顶 + 失败突破后做反转空
+ 主要趋势反转 (MTR) 后的二次入场
+ 假突破回吸 (Bull Trap) 入空场

### 1.2 不做什么

+ ❌ 反趋势空（在 AIL 状态做空）
+ ❌ 凭"感觉太高"做空
+ ❌ 单根反转 K 入场
+ ❌ 央行决议日的 JPY 系做空（干预风险）
+ ❌ Carry 不利且没有强失败动作的做空
+ ❌ Climactic 顶后 1-3 根余震期做空

### 1.3 硬性约束

+ 单笔风险 ≤ **0.5% × 账户**（做多用 1%，做空降一半）
+ 日累计亏损 ≥ **1.5R** → 当日停手
+ 周累计亏损 ≥ **3R** → 周内停手
+ 连续 **3 笔做空亏损** → 暂停做空 1 周

## 三、为什么做空难一倍 — 四大不对称

### 3.1 几何不对称

| 维度 | 上涨 | 下跌 | 启示 |
|---|---|---|---|
| 速度 | 慢、阶梯 | 快、电梯 | 做空进出场容错窗口窄 50% |
| 持续性 | 趋势可走数月 | 单波下跌经常 3-5 根反弹 | 做空必须更早离场 |
| 回调 | 健康 50% 回调常见 | 经常无回调直接 V 反弹 | 等回调入空场常等不到 |
| K 线分布 | 多正态 | 高峰厚尾 | short squeeze 比 stop run 凶 |

### 3.2 概率不对称

> 长周期均值漂移向上是**股指、加密、大宗**的常态。**外汇例外**——这是做空门槛最低的市场。

### 3.3 心理不对称

| 状态 | 做多 | 做空 |
|---|---|---|
| 浮盈 | "再走走" | "赶紧跑" |
| 浮亏 | "等等就回来" | "完蛋要轧空" |
| 持仓时长 | 自然偏长 | 强迫偏短 |

### 3.4 制度不对称（外汇专属）

+ Carry 倒贴利息（做空高息货币时）
+ 央行干预方向偏支撑本币
+ 月底 / 季末 fix flow 通常买强势货币
+ 政治风险偏支持风险偏好

## 四、高概率做空场景 — 5 个 Brooks 经典

> 按"易识别 → 难识别"排序。

### 4.1 已确立熊市 + L2 反弹失败

```plain
背景: 日图明确下跌, 5min 也连续创低
形态: 第一次反弹 (L1) 失败 → 第二次反弹 L2 失败
入场: L2 信号 K (强阴) 下方 1 tick
止损: L2 高点 + 5 pips
止盈: 前低 1R / 测幅延伸 2R
胜率: 约 60%
```

> ✅ **最教科书的高概率做空。前提是熊市已经成立**。

### 4.2 楔形顶 + 失败突破

```plain
背景: 上涨已 3 推, 最后一推:
      - 推幅缩小
      - 体积衰退
      - 收盘距前高近但 wick 长
形态: 突破前高失败, 第二根 K 跌回楔形内
入场: 跌回楔形内的强阴 K 下方
止损: 楔形最高点 + 5 pips (不是失败 K 的高)
胜率: 约 65%
```

> ✅ Brooks 反复强调：**80% of breakouts fail**——楔形顶后的失败突破是这条统计的具体应用。

### 4.3 主要趋势反转 (MTR)

```plain
背景: 日图 / 4H 强势上涨段
形态: climactic 大阳 → 1-2 根强阴 → 跌破前 swing low
      → L2 反弹失败
入场: MTR 第二段下跌起点
止损: 反弹高点 + 5 pips
胜率: 约 55% (但目标可达 1:3 R)
```

### 4.4 Trading Range 上沿

```plain
背景: 明确 trading range (3+ 次触顶触底)
形态: 触上沿 + 强阴 K + 收盘中位以下
入场: 信号 K 下方 1 tick
止损: 上沿 + 5 pips
止盈: 区间中线 / 区间下沿
胜率: 约 70%
```

> ✅ **TR 上沿做空是新手最容易掌握的高概率做空**。

### 4.5 突破失败回吸 (Bull Trap)

```plain
背景: 价格突破前高 / 阻力
形态: 1-3 根内重新跌回阻力下方
入场: 跌回 K 下方
止损: 失败突破最高点 + 5 pips
胜率: 约 60% (1-3 根内必须跌回, 4+ 根不算)
```

## 五、低概率做空黑名单

> **看到这些场景一律不空**。出现一次违反 → 当日停手。

| # | 场景 | 误判后果 |
|---|---|---|
| 1 | 强趋势上涨中"看起来累了" | 立刻被止损 |
| 2 | 单根大阳后的下一根 | 1:3 R 顺利输掉 |
| 3 | 突破阻力的当根 (没等失败) | 80% 失败但不知是不是那 20% |
| 4 | "我感觉太高了" (无失败动作) | 反复被打 |
| 5 | 已下跌 8+ 根的延续 | 接刀子 |
| 6 | 央行决议前 30 min | 干预 / 鸽派瞬间逆转 |
| 7 | 做空高息货币无强信号 | 慢性失血 |
| 8 | climactic 顶后 1-3 根 | 余震期, 反弹概率高 |

> 🛑 **铁律**：永远不在没有"明显失败动作"前做空。
> Brooks: *"Don't short a bull until it shows you it's not a bull anymore."*

## 六、执行参数差异（做空 vs 做多）

> 同一套规则**不能直接镜像**。做空必须**调整六个参数**。

| 参数 | 做多默认 | 做空建议 | 原因 |
|---|---|---|---|
| 止损距离 | 1.0 × ATR | 0.7-0.8 × ATR | 下跌速度快, 错了快撤 |
| 止盈结构 | TP1 1R / TP2 2R | TP1 0.8R / TP2 1.5R | 下跌经常不到 2R 就反弹 |
| 仓位 | 风险 1% | 风险 0.5-0.7% | 心理压力 + 滑点更大 |
| 持仓时长 | 可日内 + 隔夜 | 优先 1-3 根日内平 | 隔夜空头夜里被反弹高 |
| Trailing | EMA20 跟踪 | 前根 K 高点跟踪 (更紧) | 下跌不允许 50% 回调 |
| 信号 K 标准 | body ≥ 0.4 ATR | body ≥ 0.5 ATR + close 下 30% | 做空对信号 K 强度要求更高 |

## 七、外汇做空专项检查清单

> 做空任何外汇对前**逐项 ✅**：

1. ✅ 这个对的息差方向是否对我有利？（避免"负 carry"）
2. ✅ 当前 always-in 是哪个方向？日图 / 4H / 1H 都是空？
3. ✅ 最近重要支撑在哪？目标够远？
4. ✅ 接下来 4 小时内有 ★★★★ 数据吗？
5. ✅ 这个对最近 30 天有央行干预 / 口头干预吗？
6. ✅ 做空避险货币（USD / JPY / CHF）？地缘风险来了被打死
7. ✅ 月底 / 季末 fix 期吗？（最后两个交易日的伦敦 16:00 前后 30 min）

> 任一 ❌ → 仓位减半。两项 ❌ → **不开**。

## 八、心理与纪律

### 8.1 五种情绪状态识别 + 处置

| 状态 | 触发场景 | 身体信号 | 处置 |
|---|---|---|---|
| **FOMO** | 没仓位看着跌 | 心跳加快、想点市价单 | 关图 5 min，回来再判断 |
| **贪婪** | 想取消 TP1 | "再等等" | 严守 1R 平 50%，规则不可改 |
| **恐惧** | 浮亏想提早平 | 胃部紧张、手心出汗 | 离开屏幕，让 stop 处理 |
| **报复** | 刚亏想反向 | 脑子发热、急切感 | **强制 30 min 禁开仓** |
| **Tilt** | 连亏想找回 | 视野变窄、关注 P/L | **当日停手** |

### 8.2 做空者的灵魂三问

> 这三问**贴在屏幕旁**。每天 100 次问自己：

1. **开仓前** — *"如果我现在不开这笔，明天我会后悔吗？"*
   + 不会 → ✅ 跳过——日内做空机会无限
   + 会 → ⚠️ FOMO 在作祟，强制 5 min 冷静
   + 必须 → 🛑 绝不做——"必须"是赌徒思维

2. **持仓中** — *"如果我现在没仓位，看到现在这位置，我还会做空吗？"*
   + 会，方向相同 → ✅ 持仓正确
   + 不会 → ⚠️ 收紧 trailing
   + 会，但反向 → 🛑 立即平仓

3. **平仓后** — *"我做对了什么？做错了什么？能复制吗？"*
   + 不要问"赚了多少"
   + 只问"我执行了吗"

### 8.3 三大红线

| 红线 | 触发 | 处罚 |
|---|---|---|
| 1 | 单日累计亏损 > 1.5R（做空） | 立即关电脑，明日再来 |
| 2 | 单笔未设 stop loss | 立即平仓 + 当日停手 |
| 3 | 在禁止窗口（数据前 30 min）开仓 | 当日停手 + 复盘 |

## 九、核心参数一览

| 参数 | 默认值 | 含义 | 备注 |
|---|---|---|---|
| 单笔风险 | 0.5% | 账户最大单笔风险 | 做多用 1%，做空减半 |
| 信号 K 实体 | ≥ 0.5 × ATR | 强势空头 K 阈值 | 做空更严（做多 0.4） |
| close 位置 | ≤ 30% range | 收盘必须在下 30% | |
| 止损距离 | 0.7-0.8 × ATR | ATR 倍数 | |
| TP1 / TP2 | 0.8R / 1.5R | 止盈倍数 | 比做多更早 |
| TP1 平仓比例 | 50% | 锁定一半 | |
| trailing 距离 | 0.7 × ATR | 前根 K 高点为基准 | |
| 持仓上限 | 2 hr | 5min 单子不过夜 | |
| 日亏损上限 | 1.5R | 触发当日停手 | 做空更严（做多 2R） |

| 连续亏损上限 | 3 笔 | 触发暂停做空 1 周 | 做空专用 |

## 十、风险预算与仓位管理

### 10.1 账户级

+ 月度最大回撤上限：**5%**
+ 月度回撤 > 3% → 仓位减半 + 复盘
+ 月度回撤 > 5% → 暂停做空 2 周

### 10.2 日级

+ 单日最大开仓数（做空）：**3 笔**
+ 做多 + 做空合计单日 ≤ **5 笔**
+ 单日累计亏损 1R → 后续门槛提高（评分 ≥ 7 才做）
+ 单日累计亏损 1.5R → **停手**

### 10.3 单笔级 — 按 AI 强度

| AI 一致性 | 做空风险百分比 |
|---|---|
| 三周期一致 AIS | 0.7% |
| 二周期一致（含 5min） | 0.5% |
| 仅 5min AIS | 0.3% |
| AIU | **0%（不开）** |

## 十一、执行假设

+ 滑点：信号 K 收盘后挂 Sell Stop，滑点假设 1-2 pips
+ 延迟：网络延迟 < 200ms
+ 部分成交：主要外汇对默认全部成交
+ 订单类型：**Sell Stop（优先）** 或 Limit Sell
+ 数据时段：提前 30 min 取消挂单
+ 干预风险（JPY/CHF）：USDJPY > 155 时全部 JPY 对仅日内

## 附录 B — Brooks 做空箴言

```plain
1. "Don't short a bull until it shows you it's not a bull anymore."
```
